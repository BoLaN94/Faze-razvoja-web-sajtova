Faze razvoja web sajtova

Tokom vremena iskristalisalo se nekoliko ključnih faza u razvoju web sajtova. One su sledeće:

  • prikupljanje informacija
  • planiranje
  • dizajn
  • pisanje tekstualnog sadržaja
  • kodiranje i programiranje
  • testiranje,provjera i objavljivanje
  • održavanje,praćenje i ažuriranje

Prikupljanje informacija

Početna faza u razvoju svakog web sajta jeste ona koja podrazumeva prikupljanje informacija, odnosno zahteva. Ova faza se sastoji iz istraživanja i razmatranja kako bi se dobila što jasnija slika o osnovnim osobinama web sajta koji je potrebno napraviti. Stoga su osnovni ciljevi ove početne faze razvoja web sajtova sledeći:

  • utvrđivanje svrhe web sajta (Šta)
  • određivanje osnovnih ciljeva (Zbog čega)
  • odabir ciljne grupe (Za koga)
